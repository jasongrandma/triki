Personal Wiki Project

A color-coded knowledge tree for your notes.

Triki is a static HTML, CSS, and JavaScript app that helps you organize entries inside a vertical category hierarchy with theme-ready colors. Data is saved in-browser with localStorage for a fast, backend-free workflow.
